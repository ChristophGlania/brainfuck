Brainfuck Editor

The Brainfuck Editor enhances the standard text editor with a few functions.
  • Syntax Highlighting
    The Brainfuck key characters + - < > . , [ ] are colored. All other characters are ignored and assumed to be comments being colored differently.
  • Typing
    Typing an opening bracket [ can be completed automatically by a corresponding closing bracket ].
    Indentation is automatically propagated to the next line.
  • Content Assist
    Content assist can be triggered any time (Default is CTRL-SPACE). This will show the applicable templates. Content assist can then be toggled to display the key characters.
    Note that some of the templates operate on non-Brainfuck characters preceding the trigger location as customized input to the template.
  • Error Checking
    Code is automatically checked for matching brackets.
